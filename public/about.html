About the Website

This website would not exist without the great work of:

  • The Rarebits Template!!! I use geno7's code (or my own twisted version) anywhere I independently post comics online. This is a terrific resource for anyone making comics on their own HTML websites.
  • Eggramen's Webcomic 2.0 template, which was an enormous inspiration for how to design this website. This website is the same basic layout, but is very different under the hood (I'm using CSS grids, I don't think EggRamen's template does).
